API Documentation

Get Books

Relative URL: /getBooks

Supported Methods: GET, HEAD

Returns: A JSON array of book objects.

      Example:
      [
        {
          "author": "Name",
          "country": "America",
          "language": "English",
          "link": "Link",
          "pages": 100,
          "title": "Title",
          "year": 2024,
          "genres": [],
        },
        ...
      ]

Example Usage: GET /getBooks

Get Titles

Relative URL: /getTitles

Supported Methods: GET, HEAD

Returns: A JSON array of book titles.

      Example:
      [
        "Title 1",
        "Title 2",
        ...
      ]

Example Usage: GET /getTitles

Get Author

Relative URL: /getAuthor

Supported Methods: GET

Query Params:

author (string) - required

Returns: A JSON array of books by the specified author.

      Example:
      [
      {
        "author": "Name",
        "country": "America",
        "language": "English",
        "link": "Link",
        "pages": 100,
        "title": "Title",
        "year": 2024,
        "genres": [],
      },
      ...
    ]

Example Usage: GET /getAuthor?author=Name

Get Book

Relative URL: /getBook

Supported Methods: GET

Query Params:

title (string) - required

Returns: A JSON object of the book with the specified title.

      Example:
      {
        "book": {
            "author": "Name",
            "country": "America",
            "language": "English",
            "link": "Link",
            "pages": 100,
            "title": "Title",
            "year": 2024,
            "genres": [],
        }
      }

Example Usage: GET /getBook?title=Title

Add Book

Relative URL: /addBook

Supported Methods: POST

Body Params:

title (string) - required
author (string) - required

Returns: An empty JSON object

      Example:
      { }

Example Usage: POST /addBook with body {"title": "New Book", "author": "Author Name"}

Rate Book

Relative URL: /rateBook

Supported Methods: POST

Body Params:

title (string) - required
rating (integer) - required (1-5)

Returns: An empty JSON object

      Example:
      { }

Example Usage: POST /rateBook with body {"title": "Book Title", "rating": 5}
